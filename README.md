# ntfy BB10 Client

A native BlackBerry 10 client for **ntfy**, a simple and fast push notification service. This app allows receiving push notifications continuously on BB10 without having processes terminated unlike Android apps.

## Features

- **Receive Notifications**: Get instant push notifications from any ntfy topic.
- **Authenticity**: Built with the Cascades Framework for an authentic UI and optimal performance on BB10 devices.

## Requirements

- **Rooted BB10 OS**: The device must be rooted in order to install unsigned .bar apps. Instructions on how to root can be found [here](https://www.reddit.com/r/blackberry/comments/1qzelpa/how_to_root_the_blackberry_and_run_unsigned_apps/).

## Limitations

- Can only be used with a **self-hosted ntfy server** that does **not require HTTPS**.
- Simple notification format with title and body.

## Installation

1. Download the `.bar` file from the [releases page](https://github.com/vietsman/ntfy-bb10/releases).
2. Sideload using DBBT3.1.0.1
